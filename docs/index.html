What this changes in practice

Agents stay inside their token budget, see definitions ranked above noisy mentions, and can follow transitive dependencies in a single MCP call.


The graph PageRank boost ensures that high-centrality modules (called from many places) surface first—giving agents the architectural context they need to reason correctly.


Adjacent same-file chunks are band-merged before delivery, reducing token count by 25–91% compared to naive chunked output.
