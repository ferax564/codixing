Built for AI Agents

AST-Aware Chunking

Tree-sitter parsing across 10 language families. Functions are never split in half. The cAST algorithm recursively splits and merges to produce semantically complete chunks.

Code-Aware Search

Custom tokenizer splits camelCase, snake_case, and dot.path identifiers. BM25 ranking via Tantivy means searching for "parse config" finds parseConfig, parse_config, and ParseConfigError.

Concurrent Symbol Table

DashMap-backed symbol lookup with exact match, prefix search, and fuzzy pattern matching. Thread-safe for parallel indexing with zero contention.

Incremental Indexing

File watcher with 100ms debounce detects changes instantly. Only modified files are re-parsed and re-indexed. Sub-500ms from save to searchable.

Single Binary

No JVM, no Docker, no external databases, no Python. Pure Rust with zero runtime dependencies. Just one binary that works everywhere.

111 Tests

97 unit tests + 14 integration tests covering multi-language indexing, search accuracy, chunker boundaries, and file watcher lifecycle. All passing, zero warnings.
